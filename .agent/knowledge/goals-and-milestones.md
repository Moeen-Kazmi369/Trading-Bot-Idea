# Goals and Milestones - TradingBotAI

## Current Active Phase: MVP — Module 1
> Focus: Build a reliable backtester for the Order Block strategy.

## Key Goals
1.  **Successful Data Fetching (G1):** Fetch 1m historical OHLCV data for BTC/USDT and 10+ other coins.
2.  **Validator (G2):** Accurate Order Block detection (GRG/RGR).
3.  **Backtest (G3):** Complete backtest of the Order Block strategy across multiple years of data.
4.  **Discovery (G4):** Autonomously discover the *first* new pattern via Gemini AI.
5.  **Testnet (G5):** 100 successful trades on Binance Testnet with a positive PnL.
